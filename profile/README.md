# AstraOne – OpenNebula

<img src="https://github.com/AstraOne-Cloud.png" alt="AstraOne Logo" width="260"/>

## Description
Ce dépôt contient notre projet de groupe visant à mettre en place un **cloud privé** basé sur **OpenNebula**.  
L’objectif est de déployer une plateforme fonctionnelle permettant la gestion de machines virtuelles, réseaux, stockage et services via une interface centralisée.

## Fonctionnalités
- Gestion du cycle de vie des VM  
- Interface Sunstone (web UI)  
- Réseau virtualisé (bridges / VLAN)  
- Stockage partagé (selon configuration)  
- Automatisation partielle de l’installation  
- Documentation d’architecture
